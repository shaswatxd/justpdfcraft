# JustPDFCraft 🚀
### Complete Free Browser-Based PDF Tools & Student Toolkit

**JustPDFCraft** is a premium, 100% free web application providing over 30+ tools for PDF editing and student productivity. Everything runs directly in your browser, ensuring your files never leave your device.

🌐 **Live Demo:** [justpdfcraft.xyz](https://justpdfcraft.xyz)

---

## ✨ Features

### 📄 PDF Tools
- **Merge & Split:** Join multiple PDFs or extract specific pages.
- **Compress:** Reduce file size without losing quality.
- **Edit & Annotate:** Add text, signatures, and stamps.
- **Watermark:** Protect your documents with custom text watermarks.
- **OCR:** Extract text from scanned PDFs and images.
- **Conversions:** Word to PDF, Image to PDF, HTML to PDF, and more.

### 🎓 Student Tools
- **Calculators:** CGPA, Percentage, Attendance, and Marks calculators.
- **Generators:** Resume Builder, Cover Letter Generator, Study Planner, and Time Table Generator.
- **Utilities:** QR Code Generator, Password Generator, and Unit Converter.

---

## 🛡️ Privacy First
All processing is done **locally in the browser** using JavaScript libraries like `pdf-lib`, `pdf.js`, and `Tesseract.js`. Your sensitive documents are never uploaded to any server.

---

## 🚀 Technologies Used
- **Frontend:** HTML5, CSS3 (Modern Glassmorphism Design), Vanilla JavaScript.
- **PDF Processing:** [pdf-lib](https://pdf-lib.js.org/), [pdf.js](https://mozilla.github.io/pdf.js/).
- **OCR:** [Tesseract.js](https://tesseract.projectnaptha.com/).
- **Backend/Auth:** Firebase (for Premium features).
